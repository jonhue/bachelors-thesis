Calculates cost based on a model for an optimal distribution of loads.

  • $d$ - number of dimensions
  • $e$ - number of job types
  • $objective$ - cost function to minimize w.r.t. load assignments
  • $\lambda$ - load profile
  • $t$ - time slot
